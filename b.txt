# 2. What You Should Understand Before Monday

You only need to be comfortable with **four concepts**.

## 1️⃣ Async AI Processing

They said the service will be **async**.

Meaning the flow will look like this:

```
Client sends document
        ↓
API returns jobId immediately
        ↓
Processing happens in background
        ↓
Client polls for result
```

Example flow:

```
POST /document/extract
→ returns jobId
```

Then:

```
GET /document/extract/{jobId}
→ returns JSON result
```

Why async?

Because:

* OCR + LLM processing can take seconds
* Large documents take longer

So synchronous APIs are risky.
